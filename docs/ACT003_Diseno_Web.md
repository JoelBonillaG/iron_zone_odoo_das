# Informe ACT003: Diseño del Sitio Web
**Rol:** Desarrollador Frontend
**Proyecto:** Iron Zone Gym - Odoo 18 Implementation

## 1. Estructura y CMS
Se ha utilizado el CMS nativo de Odoo 18 para la creación y estructuración de las páginas principales. La arquitectura se basa en plantillas QWeb dinámicas que permiten la inyección de estilos personalizados (Iron Engine) para garantizar una experiencia de usuario (UX) coherente con la identidad visual del gimnasio.

### Páginas Implementadas:
- **Inicio (/):** Landing page con Hero Section dinámico, propuesta de valor y Call to Action (CTA) hacia el catálogo.
- **Nosotros (/aboutus):** Sección corporativa que presenta al equipo de ingeniería y los pilares fundamentales del proyecto.
- **Contacto (/contactus):** Formulario integrado con el backend de Odoo y datos de contacto dinámicos.

## 2. Navegación
La navegación ha sido simplificada en el encabezado (Header) para una carga cognitiva baja.
- **Ruta de Navegación:** Logo (Home) -> Tienda -> Nosotros -> Contacto.
